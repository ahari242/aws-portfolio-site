Monitor

Monitor a web page for changes.

Screenshots a provided URL every couple of seconds, comparing with the last shot.

When changes are detected, the page will open in your browser.

Node.js, Express, Socket.io, PageMonitor, BodyParser, jQuery

Draw

For drawing pixel art in the browser.

Provides controls for changing block color, block count, border color, and border size.

Responsive design suitable for large and small displays.

React, JSX, ES6

Palette Jack

Generates a color palette based on a given URLs CSS rules.

Parses css files, storing rules which relate to color. Color values are quantified and displayed in order based on frequency of occurrence.

Rules are ignored if their selector uses certain keywords such as 'error' or 'warning'.

ASP.NET MVC, HTMLAgilityPack, ExCSS, jQuery, jQueryUI, SC Color, Bootstrap

Tractor

Chrome extension for bulk-downloading files from a web page.

Will queue up downloads based on parameters such as file name, extension, and size.

Options for controlling download rate and file naming scheme.

FileSaver.js, Baffle.js, jQuery, jQueryUI, Bootstrap
